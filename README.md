# Square Calculator

A simple and clean Python program to calculate the **Area** and **Perimeter** of a square using Object-Oriented Programming.

---

## ✨ Features

- Object-Oriented design with a `Square` class
- Calculates Area (`side × side`)
- Calculates Perimeter (`4 × side`)
- User-friendly command-line interface
- Input validation for menu selection

---

## 🛠️ How to Run

1. Make sure you have **Python 3** installed
2. Save the code in a file named `square_calculator.py`
3. Run the program:

```bash
python square_calculator.py
